TUTORIAL #1 ----------- Let's start with the traditional 'hello, world' example: 

from dataflows import Flow

data = [
  {'data': 'Hello'},
  {'data': 'World'}
]

def lowerData(row):
    row['data'] = row['data'].lower()

Flow(
      data,
      lowerData
).results()[0]
>>> [[{'data': 'hello'}, {'data': 'world'}]] This very simple flow takes a list of `dict`s and applies a row processing function on each one of them.
